WanderLust is a full stack travel stay listing web application inspired by modern booking platforms. It allows users to explore destinations, view listing details, and interact with listings through a clean and responsive interface.

Key Highlights:
- User authentication and profile management
- Property/listing creation and management for hosts
- Booking flow for travelers
- Favorites/wishlist feature
- Reviews and ratings support
- Organized MVC architecture with reusable EJS views

Tech Stack:
- Backend: Node.js, Express.js
- Database: MongoDB with Mongoose
- Templating/UI: EJS, CSS, Bootstrap
- Other integrations: Cloudinary (media), session/auth middleware

This project demonstrates my skills in full stack development, RESTful routing, database design, authentication flows, and building production-style web apps end-to-end.
